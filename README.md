# DSM-from-pointCloud
 
Tutorial for creating Digital Surface Model (DSM) from Airborne Laser Scanning (ALS) data using lidR package in R.

> Assess the tutorial via this [link](https://vietducng.github.io/DSM_from_pointCloud/)
